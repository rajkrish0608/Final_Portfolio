# Raj Krish — Iron Man × Robotics/AIML Portfolio

A world-class personal portfolio inspired by the Iron Man suit assembly sequence. Built with **Next.js 14**, **Three.js**, **GSAP**, and **Tailwind CSS**.

## 🚀 Vision
This portfolio is designed to look and feel like a high-tech HUD (Heads-Up Display) and robotic assembly system. It features a centerpiece 3D suit assembly, cinematic loaders, and interactive components that showcase expertise in Robotics, AI/ML, and Creative Technology.

## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **3D Engine**: Three.js + React Three Fiber
- **Animation**: GSAP 3 + Framer Motion 11
- **Scroll**: Lenis (Smooth Scroll)
- **Styling**: Tailwind CSS + Custom CSS Variables
- **Particles**: tsParticles
- **State**: Zustand

## 🔋 Core Features
- **Suit Assembly Hero**: (In Progress) 20+ separate 3D fragments flying in to form the hero identity.
- **Arc Reactor Loader**: Custom SVG draw animation with HUD-style progress.
- **Magnetic HUD Cursor**: Dynamic targeting reticle cursor.
- **System Diagnostics**: Circular HUD skill rings.
- **Mission Archive**: 3D tilt-effect project cards with scanlines.
- **Combat Record**: Holographic mission log timeline for achievements.

## 🏗 Project Structure
```text
src/
├── app/             # Next.js App Router
├── components/      # UI Components (Hero, About, Project, etc.)
├── hooks/           # Animation & Interaction hooks
├── lib/             # Core logic (GSAP setup, Three configuration)
├── data/            # Portfolio content
└── styles/          # Design tokens & global CSS
```

## 🛠 Development

```bash
# Install dependencies

# Run development server
npm run dev

# Build for production

```
